# Скатерть-самобранка

Познакомиться с основами анализа данных с использованием популярных Python-библиотек: `pandas`, `numpy` и `matplotlib`. В ходе задания студенты применят методы работы с данными на реальном примере, используя датасет с платформы Kaggle.

## Выбор и загрузка датасета

Необходимо выбирать датасет с платформы [Kaggle](https://www.kaggle.com/), который будет использоваться для анализа данных. Это может быть любой доступный датасет.
## Основные шаги анализа данных

   - **Исследование структуры данных**:  
        Используя библиотеку `pandas`, студенты должны провести первичное исследование данных, включая просмотр общих характеристик датасета, описание типов данных, а также выявление возможных проблем (например, пропущенные значения).
    - **Обработка пропущенных значений**:  
        Студенты должны определить наличие пропущенных значений в датасете и применить подходящие методы для их обработки (удаление или заполнение).
    - **Удаление дубликатов**:  
        Необходимо проверить датасет на наличие дубликатов и, если таковые имеются, удалить их.
    - **Визуализация данных**:  
        Студенты должны построить минимум два типа визуализаций данных с помощью библиотеки `matplotlib`:
        - Линейный график для отображения изменений в данных во времени или зависимости между переменными.
        - Гистограмму или столбчатую диаграмму для сравнения данных по категориям или распределениям.
    - **Фильтрация данных**:  
        Студенты должны применить фильтрацию данных для выделения интересующих подмножеств данных (например, по значению или диапазону столбцов).
    - **Сравнение данных**:  
        Необходимо сравнить несколько групп данных или переменных с помощью статистических или визуальных методов для поиска возможных закономерностей или различий.
## Отчет  

В отчете необходимо представить:
